## 1. Description

## 2. Useful crontab commands

 * To see the setup file for crontab : 
 ```
 > crontab -l
 ```

 * To define setup file for crontab : 
 ```
 > crontab crontab.file
 ```

 * To stop crontab : 
 ```
 > crontab -r
 ```
